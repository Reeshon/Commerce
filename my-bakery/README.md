# Deleen's Home Bake

This project is a React-based e-commerce website for a bakery, built with Vite.

## Features

- Product catalog
- Shopping cart
- User authentication
- Wishlist
- Checkout process

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server

## Technologies Used

- React
- Vite
- React Router
- Firebase (for authentication and database)
- React Bootstrap
- React Toastify
